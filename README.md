# osfs26Projekt

This project was prepared today (2026-03-12), comprising the first 5 open science components:

1. Preregistration
2. Code sharing
3. Data sharing
4. Reproducible analysis
5. Reproducible manuscripts

The idea behind this project, including the detailed project protocol on this GitHub repository (see osfs26Protokoll.txt), is that each student can use this as a reference project for her/his own project, extending it and changing things whenever and wherever needed. After all, there has never been and there never will be any one size fits all solution!
